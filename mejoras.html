SEO y Rendimiento

Agregar favicon MEDIA

No hay favicon definido. Crear en formatos .ico, .svg y .png, y añadir las etiquetas <link rel="icon"> en el <head>.

Agregar meta tags Open Graph y Twitter Cards MEDIA

Sin estas etiquetas, al compartir la URL en redes sociales no se muestra una preview con imagen y descripción. Agregar: og:title, og:description, og:image, og:url, twitter:card, etc.

Alojar imágenes localmente (no depender del WP externo) MEDIA

Todas las imágenes se cargan desde agustinsantillan.com/wp-content/. Si ese hosting cae, el sitio se queda sin imágenes. Optimizar en WebP y usar srcset para responsive.

Crear robots.txt y sitemap.xml BAJA

Faltan estos archivos estándar para ayudar a los buscadores a indexar el sitio correctamente.

Implementar analytics BAJA

No hay sistema de analítica para medir visitas y comportamiento. Considerar Google Analytics o Plausible (más liviano y respetuoso con la privacidad).

UX y Contenido

Corregir errores gramaticales en textos en inglés ALTA

Correcciones principales:
• "believing in a reality?" → "dreaming of a reality?"
• "combine sensations" → "combines sensations"
• "make it really" → "make it a reality"
• "is create together" → "is to create together"
• "according your requirement" → "according to your requirements"
• "ORGANICE" → "ORGANIZE", "CONTACT WITH US" → "CONTACT US"
• "the finish project" → "the finished project"
• "what the garden...like" → "what the garden...looks like"

Diferenciar sección "How can I help you?" de "How We Work" BAJA

Ambas secciones comparten texto casi idéntico. Reescribir una de ellas para que cada sección aporte valor distinto, o fusionarlas.

Accesibilidad (a11y)

Mejorar atributos alt en imágenes de galería MEDIA

Los alt son genéricos ("Garden Project 1", "Interior Project 2"). Describir brevemente lo que se ve en cada imagen para lectores de pantalla.

Lightbox: cierre con Escape y navegación con teclado MEDIA

Verificar que el lightbox se cierre con Escape, las flechas del teclado naveguen entre fotos, y el foco quede atrapado dentro mientras esté abierto.

Agregar aria-labels a botones interactivos BAJA

Los botones del lightbox (cerrar, anterior, siguiente) y otros elementos interactivos carecen de atributos aria-label descriptivos.

Arquitectura del Código

Separar CSS y JS en archivos independientes BAJA

Todo está inline en un HTML de ~2000 líneas. Separar en styles.css y main.js para mejor mantenimiento y caché del navegador. Hacer esto cuando el proyecto crezca.
